## Network types
### (W)LAN
* Uses Ethernet cables
* Uses parallel sending
* Max range of 100 meters
* Max speed of 100 Mb/s

### WAN
* Uses fiber optics
* Uses serial sending
* Almost global range
* Max speed of 64 Mb/s
